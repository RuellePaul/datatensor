# Create a new dataset

Create your own dataset via the <a href="/datasets" title="Datasets page">datasets page</a>, then specify its name and description.

In our example, we will create a three dog species dataset: `cardigan`, `malamute` et `malinois`.

<video src="/static/images/docs/create_dataset.mp4" />

> The dataset name should reflect the set of objects you want to detect.

You will then be redirected to the newly created dataset, to proceed to the next step: adding images.
